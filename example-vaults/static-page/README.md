# example-vaults/static-page

Demo content for the bundled `static_page` theme. Mounted at `/` by
`config/config.yaml`; companion vaults are mounted at `/notes`
(`notes` theme) and `/documentation` (`notes` theme in docs mode).

Edit `index.md`, `about/index.md`, and `.leyline/vaultconfig/nav.nav` to
customise.
