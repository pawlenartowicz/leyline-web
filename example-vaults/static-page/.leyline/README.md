# Leyline vault

Files in `.leyline/vaultconfig/` are admin-only configuration: API key
hashes (`access`), allowed extensions and limits (`allowed`), optional
custom roles (`roles`) and metadata (`meta`).

Non-admin clients sync this README only; the rest of `.leyline/` is
filtered out by the server.

See https://github.com/pawlenartowicz/leyline for documentation.
