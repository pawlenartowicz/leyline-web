# Welcome

Leyline keeps a folder of plain Markdown files in sync across the tools you already use — desktop editors, a CLI daemon, or CI — and can render the same folder as a website. Open source, self-hosted.

This is the Quick Start vault — three short sections, skim in any order.

## [[1-what-is-leyline/index|1. What is Leyline?]]

Plain Markdown on disk, synced across whatever tools you reach for. The pitch in three pages.

## [[2-features/index|2. Features]]

What Leyline does, feature by feature — rendering, themes, history, and a couple of capabilities still in development.

## [[3-how-to-run-it/index|3. How to run it]]

It fits on the smallest VPS you can rent. A hosted offering is on the table if there's demand.
