# Features

What Leyline does, feature by feature.

- **[[markdown|Markdown rendering.]]** Obsidian-flavored Markdown — callouts, math, Mermaid, wikilinks, embeds, highlights, task lists, and more. Kanban rendering is coming.
- **[[themes|Themes.]]** Each vault carries its own look. Dark/light is a button.
- **[[history-and-tagging|History and tagging.]]** Edits land directly; vetted states get named tags; reverts are first-class.
- **[[editing-online|Editing in the browser.]]** *(Coming.)* The web reader grows a light editor for small fixes without a context switch.
- **[[latex-and-typst|LaTeX and Typst.]]** *(Coming.)* Collaborative `.tex` editing with PDF output; Typst → PDF export.

For the authoritative, exhaustive treatment of each, see the [[@documentation/index|full documentation]] — feature pages link the matching section.
