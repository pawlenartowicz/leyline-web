# What is Leyline?

Leyline keeps a folder of plain Markdown files in sync across whatever editors you like to use, and can render the same folder as a website.

If you've used Obsidian Sync, the elevator pitch is: open and self-hosted, with the public site built in.

Three threads:

- **[[interoperability|Plain files, your editor of choice.]]** Markdown on disk is the contract, so most tools that edit files fit in — the `leyline` CLI, Obsidian, the web reader, and any process speaking the REST API.
- **[[live-sync|Live sync, on your terms.]]** Works well for three-to-five-person teams and for solo across devices. Line-level three-way merge handles most overlap automatically; same-line conflicts get marked where you edit.
- **[[web-preview|Also a website.]]** The same folder renders as HTML through `leyline-web`, with per-file and per-user access controls kept inside the vault.
