This is an HTML file, rendered as HTML

Markdown files are rendered as Markdown; HTML files are rendered as HTML. Raw HTML embedded inside Markdown passes through unchanged — the baseline CSP (no inline <script>, no off-domain default-src, frame-ancestors 'none') is what keeps it safe.

Why HTML at all?

Some content is HTML-native — embedded media, iframes, custom layout. Treating .html as a first-class format lets vaults mix Markdown prose with HTML widgets without escape-hatch dance.
