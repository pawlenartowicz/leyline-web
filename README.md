# web

User-facing template repo for the Leyline self-hosted, read-only web view.

**Documentation:** [pawlenartowicz.pl/leyline](https://pawlenartowicz.pl/leyline)

## Versioning

This template is **0.2**. Check out the `v0.2.*` tag that matches your installed `leyline-web`
minor — a `0.2` engine needs `0.2` themes. Mixing minors (e.g. `0.2` themes on a `0.1`
engine) is unsupported and will fail to render. During the `0.x` prerelease a minor bump
(`0.2` → `0.3`) may break the engine ↔ theme contract; patch releases never do. Full policy
in the [`leyline`](https://github.com/pawlenartowicz/leyline) repo.
